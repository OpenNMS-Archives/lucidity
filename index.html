Lucidity

Fun with objects

View project onGitHub

Java object mapping made easy

Lucidity is an object mapper for Java and Cassandra. It sports a minimal set of features that are idiomatic to Java, and conform to Cassandra best-practice.

Features:

  • Annotation-based
  • One-to-many relationships
  • Inverted indexes
  • Uses CQL3 and the native protocol
  • DDL generation

Getting Started

Lucidity is not yet available from central Maven, but can be installed from the OpenNMS repository:

<project>
...
  <dependencies>
    <dependency>
      <groupId>com.opennms</groupId>
      <artifactId>lucidity</artifactId>
      <version>1.0.0-SNAPSHOT</version>
    </dependency>
  </dependencies>
...
  <pluginRepositories>
    <pluginRepository>
      <snapshots>
        <enabled>false</enabled>
      </snapshots>
      <releases>
    <enabled>true</enabled>
      </releases>
      <id>opennms-repo</id>
      <name>OpenNMS Repository</name>
      <url>http://maven.opennms.org/content/groups/opennms.org-release</url>
    </pluginRepository>
  </pluginRepositories>
...
</project>
